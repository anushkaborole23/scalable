Excellent Solution For Your Business

Traditional ML algorithms encounter great difficulties in processing the huge amount of data generated in smart production systems. This is due to the fact that they are designed under the assumption that datasets and model parameters must be entirely loaded into the memory. Scalable ML algorithms are a common way to tackle this issue, because they are well suited to handle large datasets and/or models with many parameters.

Distributed ML algorithms represent the majority of state-ofthe-art scalable ML methods [18]. They can be partitioned into two groups of algorithms that use different parallelism methods: data parallel and model parallel. In the first group, the dataset is divided into smaller pieces, which are stored on nodes in a computer cluster. All parameters of the model will be partially updated concurrently on each node and combined afterward. Conversely, in the second group, the model parameters are divided into subsets and will be updated on each node simultaneously using the whole dataset. There are also some hybrid methods where the dataset, as well as the model parameters, are split and distributed over the cluster.
