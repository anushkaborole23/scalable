Data ingestion frameworks.
Data ingestion frameworks deal with transferring raw data from data sources to the big data system and handle format and integration issues (e.g. Kafka, Flume, Sqoop, etc.). Data Storage frameworks.
Data Storage frameworks include distributed file systems and databases that persistently store varieties of big data formats.. Computation Frameworks.
Computation frameworks can process large datasets in parallel where the processing itself can be distributed across a cluster of commodity machines.
